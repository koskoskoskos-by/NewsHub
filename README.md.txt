# NewsHub — Проект лабораторной работы №2

Данный репозиторий содержит результаты выполнения лабораторной работы №2 по дисциплине «Интернет-технологии и веб-программирование». Проект включает в себя полную настройку профессионального окружения для веб-разработки и проектирование интерфейса новостного портала.

Описание проекта
Проект разделен на две основные части:

1. Infrastructure setup: Конфигурация среды разработки (VS Code, Docker, Node.js) и систем контроля версий (Git).
2. Design: Разработка визуальной концепции (мудборда) и макета веб-сайта NewsHub в графическом редакторе Figma.

Технологический стек
В ходе работы были использованы следующие инструменты и технологии:

Среда разработки и инструменты
* Visual Studio Code — основная IDE.
    * *Расширения:* Prettier (форматирование), Live Server (локальный сервер), GitLens (аналитика Git).
* Git & GitHub — система контроля версий и удаленный репозиторий для совместной работы.
* Node.js — программная платформа для работы с JavaScript на стороне сервера.
* Docker & Dev Containers — контейнеризация среды разработки для обеспечения изоляции и переносимости проекта.

Проектирование и дизайн
* Figma — основной инструмент проектирования UI/UX.
    * Использование большинства функций Figma для разработки современного дизайна
    * Применение компонентов и ассетов (Apple UI Design Resources).

Настройка окружения
Для запуска проекта в аналогичной среде необходимо:

1. Установить **Docker Desktop**.
2. Установить расширение **Dev Containers** в VS Code.
3. Подключиться к контейнеру с Node.js.
4. Создать папку в удобном месте для хранения репозитория.
5. Склонировать репозиторий:
   git clone [https://github.com/koskoskoskos-by/NewsHub](https://github.com/koskoskoskos-by/NewsHub)


Дизайн-концепция (NewsHub)
В рамках работы в Figma были созданы:
Мудборд: Сборка цветовых палитр, шрифтовых пар и визуальных референсов.
Макет сайта: Прототип главной страницы новостного портала с использованием сетки и компонентов.
